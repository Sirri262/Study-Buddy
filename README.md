# Study Buddy

Study Buddy is a simple command-line Python application to track study sessions.

## What it does
- Asks what you studied
- Asks how many minutes you studied
- Saves study sessions in a JSON file
- Loads previous sessions when the program runs again

## Technologies used
- Python
- JSON file handling

## Purpose
This project was created as a beginner-level project to practice:
- user input
- lists and dictionaries
- reading from and writing to files

## How to run
```bash
python main.py
